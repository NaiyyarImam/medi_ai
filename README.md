# Medi_AI – AI-Powered Healthcare Assistant with 3D Anatomy Visualization

Medi_AI is an intelligent healthcare assistance platform that combines AI-driven symptom analysis with interactive 3D anatomical visualization to help users better understand potential medical conditions and human body structures.
The project is designed to provide an intuitive healthcare experience by integrating modern web technologies, AI-powered analysis, and medical visualization into a single platform.

## Key Features

- AI-powered symptom analysis
- Interactive 3D anatomy visualization
- FastAPI REST API backend
- Modern Next.js frontend
- Structured medical data processing
- Scalable architecture for future healthcare integrations
- Secure API-driven communication between frontend and backend

## Installation

### Clone Repository

git clone https://github.com/NaiyyarImam/medi_ai.git

cd medi_ai

### Backend Setup

cd backend

pip install -r requirements.txt

uvicorn main:app --reload

### Frontend Setup

cd frontend

npm install

npm run dev
